StandMup API Documentation

Welcome to the documentation for StandMup API. Here's how to use it:

Getting all the videos:

URL: http://www.standmup.com/api/videos/all

Method: GET

Description: Returns a list of all videos.

Parameters:

Example:

GET https://standmup.com/videos/all

Getting video by id:

URL: http://www.standmup.com/api/videos/video_id

Method: GET

Description: Returns one video.

Example:

GET www.standmup.com/api/videos/id

Adding new video:

URL: http://www.standmup.com/api/videos

Method: POST

Description: Adds a new video.

Example:

POST www.standmup.com/api/videos

      {
            "comedian_id": "id",
            "title": "title",
            "link": "link",
            "tag_name": "tag_name",
            "description": "description"
            }

Getting a random video:

URL: http://www.standmup.com/api/random

Method: GET

Description: Returns a random video.

Example:

GET https://standmup.com/api/random

Deleting videos:

URL: http://www.standmup.com/api/videos/video_id

Method: DELETE

Description: Deletes a video by id.

Example:

DELETE https://standmup.com/api/videos/id

Getting all comedians:

URL: http://www.standmup.com/api/comedians/all

Method: GET

Description: Returns all comedians with id and name.

Example:

GET https://standmup.com/api/comedians/all

Getting all comedians by video count:

URL: http://www.standmup.com/api/comedians

Method: GET

Description: Returns all comedians with their video count.

Example:

GET https://standmup.com/api/comedians

Getting videos by comedian id:

URL: http://www.standmup.com/api/comedians/comedian_id/videos

Method: GET

Description: Returns all videos that belong to the comedian.

Example:

GET https://standmup.com/api/comedians/id/videos

Getting videos by tag:

URL: http://www.standmup.com/api/tags/tag_id

Method: GET

Description: Returns videos by tag.

Example:

GET www.standmup.com/api/tags/id

Getting stat:

URL: http://www.standmup.com/api/stat

Method: GET

Description: Returns video, comedian, tag and video by comedian counts.

Example:

GET www.standmup.com/api/stat
